# Архив диалога

Это архив видимой части нашей переписки по теме оптимизации рабочего пространства и локального голосового ассистента для ПК.

Состав:
- `01_dialog_visible.md` — реконструированная текстовая копия диалога.
- `02_project_summary.md` — краткая выжимка решений и финальной архитектуры.
- `03_assistant_setup_checklist.md` — чеклист настройки ассистента.
- `04_final_config_template.json` — шаблон `config.json` для офлайн-ассистента.

Ограничение: это не системный экспорт ChatGPT, а архив, собранный из видимого контекста переписки. Скрытые служебные инструкции, внутренние рассуждения, результаты инструментов и содержимое canvas-документа, которое не отображается в чате как обычный текст, не включены.
