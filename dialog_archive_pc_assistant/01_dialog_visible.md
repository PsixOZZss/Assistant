## 1. Оптимизация пространства на компьютере

**Пользователь:** как оптимизировать пространство на компьютере и быстро в нём ориентироваться

**Ассистент:** предложил базовую систему:
- сначала понять, что занимает место через настройки хранилища или анализаторы диска;
- удалить очевидный мусор: загрузки, корзину, старые установщики, дубликаты, кэши мессенджеров;
- сделать простую структуру папок: Работа, Учёба, Проекты, Документы, Фото и видео, Финансы, Личное, Архив, Временное;
- использовать даты в названиях папок и файлов;
- держать “Загрузки” как временную зону;
- использовать поиск Windows/macOS;
- называть файлы понятно;
- закрепить только нужные папки в быстром доступе;
- архивировать старое;
- использовать облако как резервную копию;
- делать еженедельную мини-уборку.

## 11. Озвучка и подтверждения опасных действий

**Пользователь:** а как добавить озвучку, например для подтверждения опасных действий

**Ассистент:** предложил использовать `pyttsx3` для офлайн-озвучки и двухэтапное подтверждение:

```text
1. Ты говоришь команду.
2. Ассистент понимает, что действие опасное.
3. Ассистент вслух объясняет, что собирается сделать.
4. Ждёт: “подтверждаю” или “отмена”.
5. Только после подтверждения выполняет.
```

Добавлены идеи:
- `speak()`;
- `ask_voice_confirmation()`;
- pending action;
- `prepare_sort_downloads()`;
- `confirm_sort_downloads()`.

Опасные действия:
- массовое перемещение;
- сортировка;
- очистка временных файлов;
- перенос в карантин.

Удаление предложено не реализовывать напрямую.

## 13. Уточнение про Porcupine: нужна бесплатная офлайн-альтернатива

**Пользователь:** получается работает онлайн и за плату? нужна бесплатная офлайн альтернатива, если это так

**Ассистент:** подтвердил, что Porcupine работает локально, но требует AccessKey/аккаунт, поэтому для полностью бесплатной офлайн-схемы лучше убрать Porcupine.

Новый пресет:

```text
Vosk wake phrase
Vosk command recognition
Ollama qwen3:8b
Python router
PowerShell
Everything
VS Code
pyttsx3
```

То есть:
```text
Микрофон
↓
Vosk постоянно слушает короткие фразы
↓
если услышал “компьютер” / “ассистент” / “джарвис”
↓
Vosk слушает полноценную команду
↓
Ollama переводит в JSON

```

Из установки убирается:
```powershell

```

Остаётся:
